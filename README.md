# LoginDemo
A simple demo about user manager (login/logout/change psw/isLogin/register/forget psw/etc)

一个简单的关于用户注册登录的小demo，比较简单。
使用时需要向服务器开发人员确认具体的接口，或者自己本地搭一个简单的服务器。
